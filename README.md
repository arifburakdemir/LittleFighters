# LittleFighters

## Aim of Project
In this project 2D Rougle Like game developed without using any game engines. Main challenge of this project is develop an program in high flexibility and readability by using Design Patterns. The other challenge is experience developin 2d PhysÄ±cs without using any game engines. 

## Game Play
Player tries to reach tent on the map while avoiding tarps and obstacle. Also players can create new map and place components in the way they want. They can save map and play the game on  new map. While creating map, components type with cordinates save to json file. While generating map that json file use.
